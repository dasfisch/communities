== Description ==

Adds a "Quote" link to each post, which inserts its content (inside of a &lt;blockquote&gt;) to the reply textfield.
Developed for: http://jmonkeyengine.org

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the 'bbpress-direct-quotes' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
